# Customer Chatbot

A Python Flask chatbot with NLTK, designed for customer service interactions and deployed on Render.

## 🚀 Live Demo
[https://customer-chatbot-f5vo.onrender.com](https://customer-chatbot-f5vo.onrender.com)

## 📂 Project Structure
- `app.py` → Flask entry point  
- `chatbot.py` → Chatbot logic (NLTK)  
- `requirements.txt` → Dependencies  
- `Procfile` → Deployment instructions for Render  
- `.gitignore` → Ignored files (`venv`, `__pycache__`, etc.)

## ⚙️ Setup Instructions

Clone the repository and install dependencies:
```bash
git clone https://github.com/skaim1307/customer_chatbot.git
cd customer_chatbot
pip install -r requirements.txt
